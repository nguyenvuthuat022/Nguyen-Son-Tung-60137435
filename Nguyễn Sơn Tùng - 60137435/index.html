What is AJAX

As defined in Wikipedia

AJAX is a group of interrelated web developer techniques used to creat interactive web applications or rich internet applications. With Ajax, web applications can retrieve data from the server asynchoronously in the background without interesting with the display aand behavior of the existing page

Brief History

Year Event
1995 Java applets were introduced - the first way to load data from client side asynchoronously
1996 IFrame element were introduced, which also have this functionaly
1999 Microsoft created the XMLHTTP ActiveX control in Internet Explorer 5 which then followed by other browser as XMLHttpRequest object
2005 The term "Ajax" was borned
2006 The W3C, in order to do make it official web standard, publicized the first draft specification for XMLHttpRequest Object
